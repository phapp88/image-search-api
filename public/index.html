Free Code Camp Image Search Microservice

User Stories:

  • I can get the image URLs, alt text, and page URLs for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Example image search URL: https://image-search1.herokuapp.com/api/imagesearch/lolcats%20funny?offset=10

To browse recent search queries, go to https://image-search1.herokuapp.com/api/latest/imagesearch
